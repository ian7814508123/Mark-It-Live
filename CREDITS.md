# 第三方套件與致謝 (Credits)

Markdown Live Previewer 的開發離不開開源社群的貢獻。我們在此向以下第三方專案及其貢獻者表達誠摯的感謝。
本專案使用到了以下開源套件，我們依照各開源協議的要求，在此提供相關的授權聲明。

- **[Vite](https://vite.dev)** - MIT License
- **[Tailwind CSS](https://tailwindcss.com)** - MIT License
- **[TypeScript](https://www.typescriptlang.org/)** - Apache License 2.0

## 編輯器與解析
- **[CodeMirror](https://codemirror.net)** - MIT License
- **[React Markdown](https://github.com/remarkjs/react-markdown)** - MIT License
- **[Remark](https://github.com/remarkjs/remark)** & **[Rehype](https://github.com/rehypejs/rehype)** plugins - MIT License

## 圖表與渲染
- **[Mermaid](https://mermaid.js.org)** - MIT License
- **[MathJax](https://www.mathjax.org)** - Apache License 2.0
- **[Vega](https://vega.github.io) / [Vega-Lite](https://vega.github.io)** - BSD-3-Clause License
